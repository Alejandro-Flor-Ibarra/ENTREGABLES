## 🚀 Sobre el Proyecto
El proyecto consta de una **página de inicio central (Dashboard)** que actúa como índice principal. Desde esta interfaz, se puede navegar de forma interactiva y redireccionar a la cantidad exacta de ejercicios prácticos solicitados a lo largo del programa de formación.

### 🛠️ Tecnologías Utilizadas
El desarrollo se realizó aplicando estándares web modernos y diseño limpio:
* **HTML5:** Estructura semántica del sitio.
* **CSS3:** Estilos, maquetación y diseño responsivo.
* **JavaScript:** Lógica de redirección e interactividad.

### 💻 Entorno de Desarrollo
* **Editor de Código:** [Visual Studio Code](https://visualstudio.com)

---

## 🛠️ Cómo Visualizar los Ejercicios
1. Clona este repositorio en tu computadora o descarga el archivo ZIP.
2. Abre el archivo principal `index.html` en tu navegador web preferido.
3. Utiliza el menú de la página inicial para navegar entre los diferentes ejercicios prácticos.
